# 废弃医院地点设定

## 地点背景

灾变初期的异常病例被集中送往这里。封控在极短时间内失败，留下大量中断中的救治痕迹。

## 区域功能

适合作为玩家第一次确认“污染并非传统感染”的关键调查地点。

## 氛围与视觉线索

- 被反复擦写的白板
- 永远停在凌晨的走廊挂钟
- 消毒水气味和锈味混在一起

## 资源与危险

这里有早期病历、药械仓库和隔离区残留物，也有高密度认知干扰。

## 驻留关系

陈医生曾在这里值班，他对其中一间封闭病房始终回避提及。
