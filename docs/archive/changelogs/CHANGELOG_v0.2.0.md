# CDC Survival Game - 更新日志 v0.2.0

## 🎉 新增功能

### 1. 任务系统 (QuestSystem)
- ✅ 5个任务：生存第一课、初次探索、僵尸猎人、寻找药品、生存挑战
- ✅ 任务进度追踪
- ✅ 任务奖励（物品、经验、技能点、解锁区域）
- ✅ 任务前置条件
- ✅ 任务UI界面（按 Q 键打开）

### 2. 新场景 - 废弃医院
- ✅ 医院场景（比街道更危险，50%遭遇率）
- ✅ 医疗室搜索（急救包、绷带）
- ✅ 药房搜索（止痛药、抗生素）
- ✅ 新敌人类型：僵尸患者、变异僵尸、僵尸医生

### 3. 背包UI改进
- ✅ 可使用的物品系统
- ✅ 急救包：恢复50 HP
- ✅ 绷带：恢复20 HP
- ✅ 水瓶：恢复30 口渴
- ✅ 罐头：恢复40 饥饿
- ✅ 止痛药：恢复25 精神

### 4. 保存系统集成
- ✅ 任务进度保存
- ✅ 任务完成状态保存

## 🎮 游戏流程改进

### 新手引导流程
```
1. 开始游戏 → 安全屋
2. 任务：第一课生存（自动接取）
   - 在安全屋睡觉
   - 奖励：食物罐头 x2
3. 任务：初次探索
   - 前往街道
   - 搜索一次
   - 奖励：水瓶
4. 任务：僵尸猎人
   - 击败3只僵尸
   - 奖励：绷带 x3，解锁医院
5. 任务：寻找药品
   - 前往医院
   - 搜索2次
   - 获得急救包
```

## 🎯 游戏操作

| 按键 | 功能 |

| F5 | 运行游戏 |

| I | 打开背包 |
| ESC | 返回/关闭界面 |
