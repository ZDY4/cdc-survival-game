# CDC Survival Game v0.6.0 - 制作系统与剧情更新

## 🎉 新增内容

### 1. 🔨 制作系统 (CraftingSystem)

#### 配方分类
| 分类 | 数量 | 说明 |
|------|------|------|
| **医疗** | 2 | 绷带、急救包 |
| **武器** | 4 | 小刀、棒球棍、管钳、砍刀 |
| **弹药** | 4 | 手枪弹、霰弹、步枪弹、燃料 |
| **工具** | 2 | 工具箱、开锁器 |
| **基地** | 4 | 路障、发电机、集水器、种植箱 |

#### 制作特性
- ✅ 需要收集材料
- ✅ 需要工作台（部分）
- ✅ 制作需要时间
- ✅ 队列系统（可同时排队多个）
- ✅ 可取消制作（退还材料）

#### UI界面
- 按 **C** 键打开制作面板
- 分类标签快速筛选
- 实时显示材料需求
- 制作进度条
- 制作队列管理

### 2. 📜 新剧情任务 (6个)

| 任务 | 描述 | 奖励 |
|------|------|------|
| **武装自己** | 制作第一把武器 | 废料、新配方 |
| **弹药补给** | 制作30发手枪弹 | 手枪、400经验 |
| **基地建设** | 建造3个设施 | 发电机、集水器 |
| **传说武器** | 制作武士刀 | 1000经验、称号 |
| **噩梦猎手** | 击败5个巨型变异体 | 电锯、2000经验 |
| **最终防线** | 生存10天+击败50敌人 | 突击步枪、3000经验 |

### 3. 📖 剧情大纲

#### 五章主线
1. **觉醒** - 学习生存基础
2. **探索** - 发现世界真相
3. **势力** - 遭遇其他幸存者
4. **深渊** - 深入地铁站
5. **真相** - 揭开CDC阴谋

#### 多个结局
- 结局A：逃离城市
- 结局B：揭发真相
- 结局C：牺牲自己
- 结局D：成为统治者
- 结局E：独狼传奇

#### NPC角色
- **守望者** - 神秘广播者
- **医生陈** - 前CDC研究员
- **商人老王** - 武器商人
- **小明** - 失踪的孩子
