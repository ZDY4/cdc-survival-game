# CDC Survival Game v0.3.0 - 深度更新

## 🎉 重大更新内容

### 1. 🧟 多样化敌人系统 (EnemyDatabase)

**8种不同类型的敌人：**

| 敌人 | 等级 | 特点 | 掉落 |
|------|------|------|------|
| **行尸** | 1 | 普通僵尸，缓慢移动 | 废金属 |
| **奔袭者** | 2 | 速度快，难对付 | 废金属、水瓶 |
| **巨力尸** | 3 | 血厚攻高，会眩晕 | 废金属、绷带、食物 |
| **变异体** | 4 | 会酸液喷吐、再生 | 急救包、解毒剂 |
| **医生僵尸** | 2 | 会治疗附近僵尸 | 绷带、止痛药 |
| **拾荒强盗** | 2 | 人类敌人，装备武器 | 废金属、小刀 |
| **强盗头目** | 5 | BOSS级，会召唤增援 | 钥匙、大量物资 |
| **变异犬** | 2 | 群体狩猎，速度快 | 腐肉 |

**特殊能力：**
- 眩晕攻击
- 酸液喷吐（持续伤害）
- 再生（每回合回血）
- 治疗同伴
- 呼叫增援
- 群体狩猎（有同伴时更强）

**弱点系统：**
- 头部：爆头双倍伤害
- 火焰：对僵尸有效
- 腿部：减缓速度
- 毒素：对变异敌人无效

### 2. 🍖 深度生存系统 (SurvivalSystem)

#### 核心需求：
- **饥饿** - 随时间减少，影响生命恢复
- **口渴** - 消耗更快，脱水会快速扣血
- **疲劳** - 影响战斗表现
- **体温** - 新系统！

#### 体温系统：
```
环境温度 + 时间 + 天气 + 潮湿 = 体温变化

环境温度影响：
- 晴朗: 0°C
- 雨天: -5°C
- 暴风雨: -8°C
- 雾: -3°C

时间影响：
- 深夜 (22-4点): -5°C
- 清晨 (5-7点): -3°C
- 正午 (12-15点): +5°C

体温过低: 每秒扣血
体温过高: 每秒扣血
```

#### 疾病系统：
- **流感** - 增加消耗，降低属性
- **食物中毒** - 剧烈口渴
- **伤口感染** - 战斗后可能获得
- **辐射病** - 严重，需要特殊治疗

#### 消耗计算：
```
基础消耗 + 活动消耗 + 疾病加成

活动类型：
- 静止: 基础消耗
- 移动: +50%消耗
- 战斗: +100%消耗
```

### 3. ⚔️ 深度战斗系统 (CombatSystem)

#### 战斗机制：
- **回合制** - 玩家和敌人轮流行动
- **连击系统** - 连续攻击伤害递增（最高5层）
- **防御姿态** - 减少50%伤害
- **部位打击** - 弱点部位伤害加成

#### 攻击类型：
| 类型 | 伤害 | 特点 |
|------|------|------|
| 普通攻击 | 100% | 标准伤害 |
| 重击 | 150% | 伤害高，可能被躲避 |
| 快速攻击 | 70% | 伤害低，命中高 |
| 爆头 | 200% | 击中弱点，双倍伤害 |

#### 战斗状态：
- **眩晕** - 跳过一回合
- **防御** - 伤害减免
- **增益/减益** - 属性变化
- **连击** - 持续攻击加成

#### AI行为模式：
- **被动型** - 低血量时防御
- **好战型** - 频繁使用特殊技能
- **领地型** - 满血时更激进

#### 战斗奖励：
- 经验值
- 物品掉落
- 任务进度
