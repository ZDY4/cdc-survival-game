# CDC Survival Game v0.7.0 - 装备系统更新

## 🎉 新增内容

### 1. 🛡️ 装备系统 (EquipmentSystem)

#### 装备槽位 (8个)
| 槽位 | 数量 | 示例 |
|------|------|------|
| **头部** | 3种 | 简易/军用/战术头盔 |
| **身体** | 6种 | 布衣/皮夹克/金属/战术/重型/防化服 |
| **手部** | 3种 | 皮/战术/动力手套 |
| **腿部** | 2种 | 牛仔裤/战术裤 |
| **脚部** | 3种 | 运动鞋/战斗靴/战术靴 |
| **背部** | 3种 | 小/中/大背包 |
| **饰品** | 3种 | 幸运戒指/生命护符/生存手表 |

#### 属性系统 (18种)
- **基础**: 防御、保暖、速度
- **战斗**: 精准、暴击、近战伤害、伤害减免、爆头防护
- **生存**: 生命、体力、饥饿/口渴效率、辐射/疾病抗性
- **实用**: 背包、弹药、装填速度、噪音减少

#### 特殊效果
- ✅ 爆头防护 (军用头盔)
- ✅ 夜视 (战术头盔)
- ✅ 辐射抗性 (防化服)
- ✅ 背包扩展 (战术裤、背包)
- ✅ 移动静音 (战术靴)
- ✅ 生存信息 (生存手表)

### 2. 🔧 耐久度系统

- 受到伤害时装备耐久减少
- 耐久低时显示警告（黄色/红色）
- 耐久归零装备损坏，自动卸下
- 可以使用材料修复装备
- 饰品不消耗耐久

### 3. 💻 装备UI

- 按 **E** 键打开装备面板
- 8个装备槽位可视化
- 属性总览面板
- 装备背包管理
- 一键修复功能

## 🎯 装备搭配策略

### 生存型
- 防化服 + 战术靴 + 生存手表
- 高保暖、高抗性、低消耗

### 战斗型
- 重型护甲 + 动力手套 + 军用头盔
- 高防御、高伤害、高防护

### 探索型
- 战术背心 + 大背包 + 运动鞋
- 高机动、大容量、快装填

### 潜行型
- 皮夹克 + 战术靴 + 幸运戒指
- 静音移动、高暴击

## 🔗 系统集成

### 与战斗系统
```
受到伤害 → 计算防御减免 → 减少装备耐久 → 显示伤害
```

### 与生存系统
```
环境温度 → 保暖属性 → 体温变化
饥饿/口渴 → 效率属性 → 消耗速度
```

### 与背包系统
```
装备背包/战术裤 → 增加格子 → 更大容量
```
