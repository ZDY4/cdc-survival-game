# CDC Survival Game v0.4.0 - 扩展更新

## 🎉 新增内容

### 1. 🏪 新地点 (3个)

#### 超市 (Supermarket)
- **危险等级**: 2/5
- **特点**: 可以找到大量食物和水
- **搜索点**: 货架、冷冻柜、经理室
- **敌人**: 行尸、拾荒强盗、变异犬
- **解锁条件**: 完成"初次探索"任务

#### 工厂 (Factory)
- **危险等级**: 4/5
- **特点**: 大量制作材料、废料
- **搜索点**: 车间、仓库、更衣室
- **敌人**: 巨力尸、变异体、强盗
- **解锁条件**: 完成"僵尸猎人"任务

#### 地铁站 (Subway)
- **危险等级**: 5/5 (最高)
- **特点**: 最危险的区域，但奖励最丰厚
- **搜索点**: 站台、列车、控制室
- **敌人**: 奔袭者、变异体、强盗头目
- **解锁条件**: 完成"资源收集"任务

### 2. 📜 新任务 (5个)

| 任务 | 描述 | 奖励 |
|------|------|------|
| **食物短缺** | 在超市收集10罐食物 | 5瓶水、300经验、2技能点 |
| **资源收集** | 在工厂收集20个废料 | 工具箱、400经验、解锁地铁站 |
| **地下探险** | 探索地铁站，击败5个敌人 | 3个急救包、800经验、3技能点 |
| **寻找幸存者** | 获得地铁钥匙卡 | 大量食物和水、1000经验、称号 |
| **僵尸克星** | 击败所有僵尸类型各3只 | 武器、1500经验、5技能点 |

### 3. 🧟 新敌人 (3种)

#### 巨型变异体 (Boss)
- **等级**: 6
- **HP**: 120
- **伤害**: 15
- **特殊能力**: 地面猛击、孢子云、再生
- **掉落**: 大量物资、武器、钥匙
- **出现地点**: 地铁站、工厂

#### 夜行感染者
- **等级**: 3
- **HP**: 35
- **特点**: 夜间更强，速度快
- **弱点**: 光、火
- **出现地点**: 街道B、地铁站

#### 辐射僵尸
- **等级**: 4
- **HP**: 45
- **特点**: 靠近会受到辐射伤害
- **抗性**: 辐射、毒素
- **出现地点**: 工厂、地铁站

### 4. 🎮 新特殊能力

- **地面猛击**: 范围伤害+眩晕
- **孢子云**: 持续中毒伤害
- **夜视**: 夜间命中率提升
- **无声移动**: 难以察觉
- **辐射光环**: 每回合造成辐射伤害
- **毒触**: 攻击有几率中毒

## 🎯 游戏流程

1. **新手期**: 安全屋 → 街道A → 完成任务
2. **发展期**: 解锁医院、超市，收集资源
3. **挑战期**: 前往工厂、街道B
4. **终局期**: 征服地铁站，寻找幸存者
