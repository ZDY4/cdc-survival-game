### 2. 🎯 特殊效果系统

#### 近战效果
- **流血**: 持续3-6回合伤害
- **眩晕**: 敌人跳过1回合
- **破甲**: 降低敌人防御
- **顺劈**: 攻击附近多个敌人
- **斩首**: 低血量敌人即死
- **恐惧**: 敌人可能逃跑

#### 远程效果
- **爆头加成**: 2-3倍伤害
- **散射**: 命中3个目标
- **击退**: 推开敌人
- **穿甲**: 无视50%防御
- **3连发**: 一次攻击3发

### 3. 🎮 弹药系统

| 弹药 | 用途 | 制作 |
|------|------|------|
| 石子 | 弹弓 | 免费 |
| 手枪弹 | 手枪 | 废金属+电子元件 |
| 霰弹 | 霰弹枪 | 废金属 |
| 步枪弹 | 步枪 | 废金属+电子元件 |
| 燃料 | 电锯 | 废金属+电子元件 |

### 4. 🔧 武器制作

- 收集材料制作武器
- 使用工具箱加速
- 高级武器需要更多材料
- 武器可以修复

### 5. 💻 武器UI

- 按 **W** 键打开武器面板
- 查看当前武器状态
- 装备/切换武器
- 修复损坏的武器
- 查看弹药数量

## 🎮 战斗流程改进

### 旧版
```
攻击 → 固定伤害(5-15) → 敌人反击
```

### 新版
```
选择武器 → 检查弹药 → 计算伤害 
→ 暴击判定 → 特殊效果 → 减少耐久
→ 敌人反击
```

## 🎯 游戏策略

### 早期游戏
- 使用小刀/棒球棍
- 弹弓远程辅助
- 保存弹药

### 中期游戏
- 制作管钳/砍刀
- 使用手枪/霰弹枪
- 收集材料

### 后期游戏
- 武士刀/电锯近战
- 步枪精准打击
- 突击步枪火力压制
