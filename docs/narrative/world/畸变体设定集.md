# 相移畸变体设定集
## 统一设定基础
所有畸变体均为相移污染下，不同种类的有机物（人类/动物血肉、植物）和无机物（金属、混凝土、塑料、电子元件）发生随机坍缩融合的产物：
- 无固定物种分类、无统一生理结构、无自我意识，仅遵循「攻击所有未被污染活物」的本能
- 融合物质种类越多、形态越复杂，对应污染等级越高，对理智的侵蚀能力越强
- 完全没有传统怪物的生物特征，形态不可名状，主打违和感带来的深层克苏鲁恐惧

## 轻度污染区畸变体（融合程度低，仅机械+少量血肉融合）
### 1. 街区杂糅体
- **出现区域**：CXL路周边普通街区、据点周边道路
- **融合构成**：共享单车金属车架、快递柜铁皮、路边垃圾桶塑料外壳 + 流浪猫/老鼠/飞虫血肉
- **形态描述**：半人高的不规则金属肉块，到处都是外露的自行车链条、垃圾桶滚轮、猫爪/鼠牙，移动时发出金属摩擦和血肉挤压的混合怪声
- **攻击方式**：甩动金属链条抽打、喷射带血的锈水、用外露的牙齿撕咬
- **理智影响**：单次直视损失1点理智，几乎无影响
- **弱点**：金属外壳很薄，新手工兵铲3-4下即可击碎核心

### 2. 路灯光影畸变体
- **出现区域**：所有有路灯的道路区域
- **融合构成**：路灯金属灯杆、高压线路、LED灯管 + 大量趋光飞虫/蝙蝠血肉
- **形态描述**：2-3米高的扭曲灯柱状，灯管部分和飞虫复眼融合，不断发出明暗闪烁的诡异光线，表面爬满半机械半血肉的飞虫
- **攻击方式**：照射刺眼闪光造成短暂失明、释放低压电流麻痹、操控飞虫群叮咬
- **理智影响**：直视闪光单次损失3点理智
- **弱点**：攻击发光的灯管核心即可快速摧毁

## 中度污染区畸变体（融合程度高，大量有机物+无机物混合）
### 1. 公交坍缩体
- **出现区域**：城市主干道、公交站周边
- **融合构成**：整辆公交车金属外壳、座椅、轮胎、油箱 + 灾变时车内乘客、司机的全部血肉
- **形态描述**：3-5米高的不规则巨型肉块，车身外壳和血肉完全揉合在一起，车窗位置长着大量人类眼睛，轮胎和人类腿脚融合，移动时发出公交车发动机轰鸣和人类哀嚎的混合声音
- **攻击方式**：甩动变形的车门拍打、喷射燃烧的柴油和血肉混合物、碾压靠近的活物
- **理智影响**：持续直视每秒损失2点理智
- **弱点**：攻击油箱位置可造成爆炸，直接摧毁

### 2. 街面聚合体
- **出现区域**：商业街、商铺密集区域
- **融合构成**：商铺广告牌、空调外机、卷帘门、玻璃橱窗 + 躲在商铺内的幸存者、宠物血肉
- **形态描述**：沿着街面铺开的不规则血肉层，上面嵌着各种变形的招牌、空调外机、碎玻璃，不断发出错乱的广告配音、叫卖声、人类哭喊的混合声音
- **攻击方式**：喷射碎玻璃划伤、甩动卷帘门碎片切割、发出高分贝噪音造成眩晕
- **理智影响**：听到声音每秒损失1点理智，直视形态每秒损失3点理智
- **弱点**：攻击嵌在内部的广告牌电源模块即可瘫痪

### 3. 闸机嵌合体
- **出现区域**：M1线各站点入口、地铁站厅
- **融合构成**：地铁闸机金属机身、刷卡模块、金属挡杆 + 灾变时逃灾的乘客血肉
- **形态描述**：1.5米高的类人型畸变体，闸机外壳和人类躯干融合，手臂是变形的金属挡杆，头部是刷卡区域和人类面部的融合体，移动时不断发出「请刷卡、请通行」的错乱电子音
- **攻击方式**：伸金属挡杆穿刺、用闸机金属机身撞击、发出电子音造成短暂麻痹
- **理智影响**：直视每秒损失2点理智
- **弱点**：攻击头部的刷卡模块核心即可摧毁

## 重度污染区畸变体（融合程度极高，意识层污染附加，完全不可名状）
### 1. 隧道延伸畸变体
- **出现区域**：M1线隧道深处、TZL站核心区周边
- **融合构成**：整列M1线列车、隧道混凝土/钢筋、轨道、电缆 + 灾变时列车上千名乘客的全部血肉
- **形态描述**：没有头尾的巨型长条混沌体，沿着隧道不断延伸，表面嵌着列车座椅、车窗、钢筋、人类肢体，没有固定的移动方向，像活的一样不断蠕动伸长，经过的地方会留下带血的金属碎屑
- **攻击方式**：甩动外露的钢筋穿刺、喷射被污染的地下水、车身挤压碾压
- **理智影响**：直视每秒损失5点理智，长期直视会产生「自己也被融合进车身」的幻觉
- **弱点**：只有攻击列车车头位置的核心融合区才能造成伤害，普通攻击几乎无效

### 2. 地层蠕动畸变体
- **出现区域**：地面沉降严重区域、地下空间入口
- **融合构成**：高楼地基钢筋混凝土、地下岩石泥土、下水道有机物、大量被埋的幸存者/动物血肉
- **形态描述**：没有固定形状的巨型肉山，高度可达10米以上，表面不断蠕动凸起，长出各种不规则的混凝土尖刺、人类肢体、植物根茎，移动时会造成地面轻微震动
- **攻击方式**：喷射混凝土碎块、伸出肉刺穿刺、释放有毒的地下气体
- **理智影响**：靠近到10米范围内每秒损失3点理智，直视每秒损失6点理智，会产生「地面在不断往下陷要把自己吞进去」的幻觉
- **弱点**：攻击表面蠕动的发光核心点，摧毁3个以上核心即可瘫痪

### 3. 阈限幻影畸变体
- **出现区域**：所有重度污染区随机刷新
- **融合构成**：无固定实体，是大量死者意识碎片、污染能量的聚合体，仅能看到一团不断扭曲的光影
- **形态描述**：没有固定形态，就是一团不断变化颜色和形状的扭曲光影，有时会出现死者的模糊面孔，有时会变成玩家熟悉的人的样子，靠近会听到大量死者的低语声
- **攻击方式**：无物理攻击，仅通过意识侵蚀造成理智值快速下降，理智值归零会直接被污染同化
- **理智影响**：靠近到20米范围内每秒损失10点理智，完全无法通过物理手段防御
- **弱点**：无法被物理武器摧毁，仅能通过特殊的抗污染药剂、声波驱赶装置暂时驱散

## 最终BOSS：核心坍缩畸胎
- **出现区域**：M1线TZL站地下62米，污染核心原点
- **融合构成**：盾构机全部金属部件、施工队全部人员血肉、地下岩石、阈限界面溢出的未知能量
- **形态描述**：直径20米以上的巨型混沌球体，不断收缩膨胀，表面没有任何固定结构，金属、血肉、岩石完全揉合成不可名状的形态，看不到任何可识别的部件，不断发出极低频率的嗡鸣声
- **攻击方式**：释放范围意识冲击瞬间扣除大量理智、操控周围所有畸变体攻击、扭曲空间造成玩家位置错乱
- **理智影响**：进入核心区范围每秒损失20点理智，直视本体瞬间扣除50点理智，长期直视会直接精神崩溃
- **弱点**：只有用特制的污染阻隔炸弹投入核心内部才能彻底摧毁
