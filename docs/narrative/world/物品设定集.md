# 物品设定集
## 统一设定基础
所有物品均为灾变2年后幸存者利用日常物资手工改装制作，无高科技工业产品：
- 物资来源全部为灾变前城市遗留物品，无官方生产供应，稀缺度随对应原材料获取难度递增
- 所有物品设计贴合本土场景，大量使用本地常见物品改装，符合CDC城市背景
- 通用货币规则同世界观设定，标准计价单位为「镍币」，口语中「镍币」「钢镚」「硬币」全部通用，无歧义

## 一、生存消耗品
### 基础食物类
1. **压缩饼干/方便面**
   - 获取方式：外出搜集、商人老王处兑换
   - 功能：基础口粮，1份可维持1天生存需求
   - 售价：10镍币/份
2. **火锅底料**
   - 获取方式：外出搜集、任务奖励、老王处高价兑换
   - 功能：高价值硬通货，1份可兑换3天口粮/1个基础滤罐，也可煮制食物临时提升10%理智抗性2小时
   - 售价：100镍币/份
3. **盖碗茶**
   - 获取方式：据点餐饮处兑换、外出搜集瓶装
   - 功能：饮用后瞬间恢复10点理智值，缓解轻度污染带来的眩晕
   - 售价：20镍币/份
4. **冰粉/凉虾**
   - 获取方式：据点餐饮处兑换、外出搜集瓶装
   - 功能：食用后恢复20点生命值，消暑降温提升移动速度10%半小时
   - 售价：15镍币/份
5. **白酒**
   - 获取方式：外出搜集、老王处兑换
   - 功能：可用于伤口消毒，饮用后临时提升20%理智抗性1小时，也可作为燃烧瓶原料
   - 售价：50镍币/瓶
6. **本土泡菜**（农耕营专属）
   - 获取方式：农耕营兑换、任务奖励
   - 功能：食用后恢复15点生命值+10点理智值，唯一同时恢复两种属性的食物
   - 售价：30镍币/份

### 污染/理智相关消耗品
1. **基础滤罐**
   - 获取方式：张师傅处制作、外出搜集、老王处兑换
   - 功能：适配防毒面具，可在轻度污染区使用1小时，中度污染区使用20分钟
   - 售价：30镍币/个
2. **抗污染药片**
   - 获取方式：陈医生处制作、任务奖励
   - 功能：服用后可抵消10分钟的轻度污染伤害，减轻中度污染的持续损伤
   - 售价：40镍币/片
3. **长效抗污染药剂**（科技营专属）
   - 获取方式：科技营兑换、高级任务奖励
   - 功能：效果是普通药片的3倍，可在中度污染区停留2小时无损伤
   - 售价：120镍币/片
4. **应急镇静剂**
   - 获取方式：陈医生处制作、高级任务奖励
   - 功能：注射后瞬间恢复50点理智值，解除幻觉、眩晕等debuff
   - 售价：200镍币/支
5. **本土磁带**
   - 获取方式：DJ记忆区域搜集、老王处兑换
   - 功能：收录灾变前川剧、评书、本土民谣，使用播放器播放可抵消畸变体的低语影响，缓慢恢复理智值（每分钟恢复2点）
   - 售价：80镍币/盒
6. **燃烟驱虫剂**（农耕营专属）
   - 获取方式：农耕营兑换
   - 功能：燃烧后可驱散50米内所有小型畸变体
   - 售价：50镍币/份

### 通用材料类
1. **电子元件**
   - 获取方式：外出搜集、手机/电脑拆解
   - 功能：维修电子设备、改装武器的基础材料
   - 售价：5镍币/个
2. **钢筋/混凝土块**
   - 获取方式：外出拆解建筑、工程守望者处兑换
   - 功能：维修据点工事、制作重型武器的基础材料
   - 售价：8镍币/块
3. **PVC管/弹力绳**
   - 获取方式：外出搜集、五金店拆解
   - 功能：制作远程武器、陷阱的基础材料
   - 售价：10镍币/份
4. **化学品/药剂原料**
   - 获取方式：医院/实验室搜集、科技营兑换
   - 功能：制作抗污染药剂、腐蚀武器的基础材料
   - 售价：25镍币/份
5. **铅粒**
   - 获取方式：外出搜集、五金店拆解
   - 功能：制作弹弓子弹、基础防污染的通用材料
   - 售价：2镍币/粒
