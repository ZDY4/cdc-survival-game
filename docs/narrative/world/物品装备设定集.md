# 物品/武器/装备设定集
## 统一设定基础
所有物品均为灾变2年后幸存者利用日常物资手工改装制作，无高科技工业产品：
- 物资来源全部为灾变前城市遗留物品，无官方生产供应，稀缺度随对应原材料获取难度递增
- 所有装备/武器设计贴合本土场景，大量使用本地常见物品改装，符合CDC城市背景
- 物品功能完全适配相移污染、畸变体特性，区分生存、防护、战斗、特殊功能四大类

## 一、通用生存消耗品
### 基础生存类
1. **压缩饼干/方便面**
   - 获取方式：外出搜集、商人老王处兑换
   - 功能：基础口粮，1份可维持1天生存需求
   - 价值：10贡献点/份
2. **火锅底料**
   - 获取方式：外出搜集、任务奖励、老王处高价兑换
   - 功能：通用硬通货，1份可兑换3天口粮/1个基础滤罐，也可煮制食物临时提升10%理智抗性2小时
   - 价值：100贡献点/份
3. **盖碗茶**
   - 获取方式：据点餐饮处兑换、外出搜集瓶装
   - 功能：饮用后瞬间恢复10点理智值，缓解轻度污染带来的眩晕
   - 价值：20贡献点/份
4. **冰粉/凉虾**
   - 获取方式：据点餐饮处兑换、外出搜集瓶装
   - 功能：食用后恢复20点生命值，消暑降温提升移动速度10%半小时
   - 价值：15贡献点/份
5. **白酒**
   - 获取方式：外出搜集、老王处兑换
   - 功能：可用于伤口消毒，饮用后临时提升20%理智抗性1小时，也可作为燃烧瓶原料
   - 价值：50贡献点/瓶

### 污染/理智相关消耗品
1. **基础滤罐**
   - 获取方式：张师傅处制作、外出搜集、老王处兑换
   - 功能：适配防毒面具，可在轻度污染区使用1小时，中度污染区使用20分钟
   - 价值：30贡献点/个
2. **抗污染药片**
   - 获取方式：陈医生处制作、任务奖励
   - 功能：服用后可抵消10分钟的轻度污染伤害，减轻中度污染的持续损伤
   - 价值：40贡献点/片
3. **应急镇静剂**
   - 获取方式：陈医生处制作、高级任务奖励
   - 功能：注射后瞬间恢复50点理智值，解除幻觉、眩晕等debuff
   - 价值：200贡献点/支
4. **本土磁带**
   - 获取方式：DJ记忆区域搜集、老王处兑换
   - 功能：收录灾变前川剧、评书、本土民谣，使用播放器播放可抵消畸变体的低语影响，缓慢恢复理智值（每分钟恢复2点）
   - 价值：80贡献点/盒

## 二、防护装备
### 基础防护类
1. **普通医用口罩**
   - 获取方式：外出搜集、陈医生处兑换
   - 功能：基础防护，仅可在轻度污染区使用30分钟，无法阻挡意识层污染
   - 价值：5贡献点/个
2. **改装电瓶车头盔**
   - 获取方式：张师傅处改装、任务奖励
   - 功能：加装滤棉和墨镜片，可作为初级防毒面具，防护轻度污染1小时，同时阻挡畸变体闪光攻击、减少直视畸变体的理智损失30%
   - 价值：100贡献点/个
3. **麻将席防刺背心**
   - 获取方式：赵阿姨组织幸存者制作、老王处兑换
   - 功能：用废弃麻将席加帆布缝制，可抵御小型畸变体的撕咬、碎玻璃划伤，减少物理伤害20%
   - 价值：150贡献点/件
4. **外卖箱防护甲**
   - 获取方式：李快递改装、任务奖励
   - 功能：用废弃外卖保温箱加铝箔制作，轻便防水，可阻挡轻度污染腐蚀，减少化学伤害30%
   - 价值：200贡献点/件

### 高级防护类
1. **全覆式防毒面具**
   - 获取方式：外出搜集、高级任务奖励
   - 功能：专业级防护，可在中度污染区使用1小时，重度污染区使用20分钟，完全阻挡意识层污染的直接侵蚀
   - 价值：500贡献点/个
2. **隔音耳罩**
   - 获取方式：张师傅改装、任务奖励
   - 功能：用降噪耳机改装，可抵消畸变体的噪音攻击、低语影响，减少理智损失50%
   - 价值：300贡献点/个
3. **防化服**
   - 获取方式：医院/实验室区域搜集、稀有任务奖励
   - 功能：可在重度污染区使用1小时，完全阻挡物理/化学/意识层污染伤害，仅在高风险任务中使用
   - 价值：1000贡献点/件

## 三、武器
### 近战武器
1. **新手工兵铲**
   - 获取方式：初始赠送、老王处兑换
   - 功能：基础近战武器，攻击力15点，可应对小型畸变体、普通劫掠者
   - 价值：50贡献点/把
2. **电击工兵铲**
   - 获取方式：张师傅改装、任务002奖励
   - 功能：加装电瓶车电机和电池，攻击附带20点电击伤害，可暂时击晕机械类畸变体3秒
   - 价值：200贡献点/把
3. **共享单车链锤**
   - 获取方式：自行改装、张师傅处制作
   - 功能：用共享单车链条加锁头制作，攻击力25点，攻击范围大，适合应对集群小型畸变体
   - 价值：120贡献点/把
4. **碎瓷刀**
   - 获取方式：自行制作、外出搜集
   - 功能：用瓷碗/酒瓶碎片嵌在木柄上制作，锋利度高，攻击力30点，对血肉类畸变体伤害加成50%
   - 价值：100贡献点/把
5. **混凝土重锤**
   - 获取方式：自行制作、张师傅处改装
   - 功能：用钢筋加混凝土块制作，攻击力40点，攻速慢，可砸开畸变体厚重的金属外壳，对核心伤害加成100%
   - 价值：300贡献点/把

### 远程武器
1. **手工弹弓**
   - 获取方式：初始赠送、自行制作
   - 功能：发射钢珠/玻璃珠，攻击力10点，适合远距离打击小型畸变体的发光核心
   - 价值：30贡献点/把
2. **PVC弩**
   - 获取方式：张师傅改装、任务奖励
   - 功能：用PVC管加弹力绳制作，发射钢筋短箭，攻击力30点，有效射程20米，适合对付中型畸变体
   - 价值：250贡献点/把
3. **灭火器喷射器**
   - 获取方式：自行改装、张师傅处制作
   - 功能：用废弃灭火器改装，可装填汽油/消毒水/腐蚀药剂，喷射范围5米，对血肉类畸变体伤害加成80%
   - 价值：350贡献点/把
4. **广场舞喇叭声波武器**
   - 获取方式：张师傅改装、稀有任务奖励
   - 功能：用灾变前广场舞大喇叭改装，发射高频声波，可驱散阈限幻影类意识畸变体，对所有畸变体造成眩晕效果2秒
   - 价值：600贡献点/把
5. **毒针发射器**
   - 获取方式：陈医生+张师傅联合制作、高级任务奖励
   - 功能：用自行车打气筒改装，发射装填了麻醉剂/污染中和剂的毒针，可瞬间麻痹中型畸变体10秒，对小型畸变体一击必杀
   - 价值：700贡献点/把

## 四、特殊功能道具
1. **改装磁带播放器**
   - 获取方式：外出搜集、李快递处兑换
   - 功能：用旧随身听改装，可播放本土磁带，抵消畸变体低语影响，缓慢恢复理智值，是长时间外出探索的必备道具
2. **污染探测仪**
   - 获取方式：张师傅改装、任务奖励
   - 功能：用旧手机加传感器改装，可检测周围50米内的污染浓度，提前预警高污染区域和隐藏畸变体
3. **荧光标记器**
   - 获取方式：自行制作、老王处兑换
   - 功能：用荧光漆加喷壶制作，可在墙面/地面做标记，防止重度污染区空间扭曲时迷路
4. **信号弹**
   - 获取方式：外出搜集、高级任务奖励
   - 功能：发射后可在10公里外看到，遇到危险时发射，据点巡逻队会赶来支援
5. **便携隔离帐篷**
   - 获取方式：李快递改装、任务奖励
   - 功能：用外卖箱保温层加防水布制作，1分钟即可搭建完成，内部可阻隔轻度污染，供外出小队临时休息、补充状态
6. **万能工具包**
   - 获取方式：张师傅处兑换、任务奖励
   - 功能：包含螺丝刀、扳手、胶带、电线等基础工具，可临时维修装备、拆解畸变体部件获取材料
