# 武器装备设定集
## 统一设定基础
所有装备/武器均为灾变2年后幸存者利用日常物资手工改装制作，无高科技工业产品：
- 物资来源全部为灾变前城市遗留物品，无官方生产供应，稀缺度随对应原材料获取难度递增
- 所有装备/武器设计贴合本土场景，大量使用本地常见物品改装，符合CDC城市背景
- 通用货币规则同世界观设定，标准计价单位为「镍币」，口语中「镍币」「钢镚」「硬币」全部通用，无歧义

## 一、防护装备
### 基础防护类
1. **普通医用口罩**
   - 获取方式：外出搜集、陈医生处兑换
   - 功能：基础防护，仅可在轻度污染区使用30分钟，无法阻挡意识层污染
   - 售价：5镍币/个
2. **改装电瓶车头盔**
   - 获取方式：张师傅处改装、任务奖励
   - 功能：加装滤棉和墨镜片，可作为初级防毒面具，防护轻度污染1小时，同时阻挡畸变体闪光攻击、减少直视畸变体的理智损失30%
   - 售价：100镍币/个
3. **麻将席防刺背心**
   - 获取方式：赵阿姨组织幸存者制作、老王处兑换
   - 功能：用废弃麻将席加帆布缝制，可抵御小型畸变体的撕咬、碎玻璃划伤，减少物理伤害20%
   - 售价：150镍币/件
4. **外卖箱防护甲**
   - 获取方式：李快递改装、任务奖励
   - 功能：用废弃外卖保温箱加铝箔制作，轻便防水，可阻挡轻度污染腐蚀，减少化学伤害30%
   - 售价：200镍币/件

### 高级防护类
1. **全覆式防毒面具**
   - 获取方式：外出搜集、高级任务奖励
   - 功能：专业级防护，可在中度污染区使用1小时，重度污染区使用20分钟，完全阻挡意识层污染的直接侵蚀
   - 售价：500镍币/个
2. **隔音耳罩**
   - 获取方式：张师傅改装、任务奖励
   - 功能：用降噪耳机改装，可抵消畸变体的噪音攻击、低语影响，减少理智损失50%
   - 售价：300镍币/个
3. **防化服**
   - 获取方式：医院/实验室区域搜集、稀有任务奖励
   - 功能：可在重度污染区使用1小时，完全阻挡物理/化学/意识层污染伤害，仅在高风险任务中使用
   - 售价：1000镍币/件
4. **竹制防具套装**（农耕营专属）
   - 获取方式：农耕营声望尊敬兑换
   - 功能：重量轻、防御高，比普通防刺甲多30%穿刺防御
   - 售价：400镍币/套
5. **混凝土防具**（工程守望者专属）
   - 获取方式：工程守望者声望尊敬兑换
   - 功能：防御是普通防刺甲的3倍，可抵御重型畸变体的碾压攻击
   - 售价：800镍币/套

## 二、武器
### 近战武器
1. **新手工兵铲**
   - 获取方式：初始赠送、老王处兑换
   - 功能：基础近战武器，攻击力15点，可应对小型畸变体、普通劫掠者
   - 售价：50镍币/把
2. **电击工兵铲**
   - 获取方式：张师傅改装、任务奖励
   - 功能：加装电瓶车电机和电池，攻击附带20点电击伤害，可暂时击晕机械类畸变体3秒
   - 售价：200镍币/把
3. **共享单车链锤**
   - 获取方式：自行改装、张师傅处制作
   - 功能：用共享单车链条加锁头制作，攻击力25点，攻击范围大，适合应对集群小型畸变体
   - 售价：120镍币/把
4. **碎瓷刀**
   - 获取方式：自行制作、外出搜集
   - 功能：用瓷碗/酒瓶碎片嵌在木柄上制作，锋利度高，攻击力30点，对血肉类畸变体伤害加成50%
   - 售价：100镍币/把
5. **混凝土重锤**
   - 获取方式：自行制作、张师傅处改装
   - 功能：用钢筋加混凝土块制作，攻击力40点，攻速慢，可砸开畸变体厚重的金属外壳，对核心伤害加成100%
   - 售价：300镍币/把
6. **电钻穿刺武器**（工程守望者专属）
   - 获取方式：工程守望者兑换
   - 功能：用电钻改装，对金属/混凝土类畸变体伤害加成50%
   - 售价：600镍币/把

### 远程武器
1. **手工弹弓**
   - 获取方式：初始赠送、自行制作
   - 功能：发射钢珠/玻璃珠，攻击力10点，适合远距离打击小型畸变体的发光核心
   - 售价：30镍币/把
2. **PVC弩**
   - 获取方式：张师傅改装、任务奖励
   - 功能：用PVC管加弹力绳制作，发射钢筋短箭，攻击力30点，有效射程20米，适合对付中型畸变体
   - 售价：250镍币/把
3. **灭火器喷射器**
   - 获取方式：自行改装、张师傅处制作
   - 功能：用废弃灭火器改装，可装填汽油/消毒水/腐蚀药剂，喷射范围5米，对血肉类畸变体伤害加成80%
   - 售价：350镍币/把
4. **广场舞喇叭声波武器**
   - 获取方式：张师傅改装、稀有任务奖励
   - 功能：用灾变前广场舞大喇叭改装，发射高频声波，可驱散阈限幻影类意识畸变体，对所有畸变体造成眩晕效果2秒
   - 售价：600镍币/把
5. **毒针发射器**（科技营专属）
   - 获取方式：科技营声望尊敬兑换
   - 功能：用自行车打气筒改装，发射装填了麻醉剂/污染中和剂的毒针，可瞬间麻痹中型畸变体10秒，对小型畸变体一击必杀
   - 售价：700镍币/把
6. **激光笔改装枪**（科技营专属）
   - 获取方式：科技营兑换
   - 功能：对所有类型畸变体伤害加成20%
   - 售价：750镍币/把

## 三、特殊功能装备
1. **改装磁带播放器**
   - 获取方式：外出搜集、李快递处兑换
   - 功能：用旧随身听改装，可播放本土磁带，抵消畸变体低语影响，缓慢恢复理智值，是长时间外出探索的必备道具
   - 售价：180镍币/个
2. **基础污染探测仪**
   - 获取方式：张师傅改装、任务奖励
   - 功能：用旧手机加传感器改装，可检测周围50米内的污染浓度，提前预警高污染区域和隐藏畸变体
   - 售价：220镍币/个
3. **高级污染探测仪**（科技营专属）
   - 获取方式：科技营声望尊敬兑换
   - 功能：探测范围扩大到200米，可精准定位隐藏畸变体、稀有物资位置
   - 售价：600镍币/个
4. **荧光标记器**
   - 获取方式：自行制作、老王处兑换
   - 功能：用荧光漆加喷壶制作，可在墙面/地面做标记，防止重度污染区空间扭曲时迷路

5. **便携隔离帐篷**

   - 售价：300镍币/个
6. **万能工具包**
   - 获取方式：张师傅处兑换、任务奖励

   - 售价：150镍币/个
7. **工程探测仪**（工程守望者专属）
   - 获取方式：工程守望者声望友善兑换
   - 功能：可探测地下结构、隧道坍塌风险，是进入核心区隧道的必备道具
   - 售价：450镍币/个
