# 幸存者阵营设定集
## 统一设定基础
所有阵营均为灾变2年后幸存者自发形成的组织，无官方背景，每个阵营有独立的立场、生存逻辑、专属生产链，玩家可选择中立、结盟、敌对任意阵营，选择会直接影响剧情走向和可解锁结局：
- 阵营之间存在天然利益冲突，选边站会锁死部分剧情、解锁对应专属内容
- 所有阵营选址均符合污染传播规则，位于相对安全的低污染区域
- 阵营声望分为5级：「敌对」→「中立」→「友善」→「尊敬」→「崇拜」，不同等级对应不同权限、奖励

## 1. LQ山农耕营（参考：龙泉山农耕幸存者）
### 基础信息
- **选址**：LQ山东侧高地，完全不受污染影响，有大片可耕种土地
- **人员构成**：灾变前的农民、农家乐老板、普通市民，总人数约300人，是现存规模最大的幸存者阵营
- **核心理念**：保守求生，远离污染区，不掺和任何和污染研究、核心区探索相关的事，只想安稳种地活下去，自给自足，对外仅进行必要的物资交易
- **势力范围**：LQ山全域、东部郊区低污染区，对外来人员管控严格，禁止携带任何污染物品进入营地

### 专属生产链&独有物品
- 专属产出：本土特色农作物（折耳根、耙耙柑、青菜）、自酿白酒、土制化肥、家养家禽肉蛋
- 独有配方：
  1. 本土泡菜：食用后恢复15点生命值+10点理智值，是唯一同时恢复两种属性的食物
  2. 竹制防具套装：重量轻、防御高，比普通防刺甲多30%穿刺防御
  3. 燃烟驱虫剂：燃烧后可驱散50米内所有小型畸变体
- 独有武器：柴刀、竹制长矛、农用钉耙改装的近战武器，对血肉类畸变体伤害加成30%

### 声望奖励
| 声望等级 | 解锁权限 |
| --- | --- |
| 敌对 | 进入LQ山范围会被主动攻击 |
| 中立 | 可在营地门口交易区进行基础物资交易 |
| 友善 | 可接农耕营任务，可进入营地内部住宿、购买专属食物 |
| 尊敬 | 解锁所有独有配方，可兑换农耕营专属防具套装 |
| 崇拜 | 解锁「隐居结局」：可留在LQ山成为正式营民，安稳生存不再需要外出冒险，解锁专属农耕经营玩法 |

### 剧情关联
- 主线关键节点：农耕营掌握着仅有的安全撤离路线，想走「逃离CDC」结局必须获得农耕营的信任拿到通行权限
- 冲突点：其他阵营想征用农耕营的土地建防御工事，玩家需要选择帮农耕营守住土地，还是帮其他阵营说服农耕营合作

## 2. GXU科技营（参考：城西高校集群幸存者）
### 基础信息
- **选址**：城西高校集群高地，实验室、教学楼密集，易守难攻
- **人员构成**：灾变前的高校教授、学生、科研人员、技术工人，总人数约150人，是技术水平最高的阵营
- **核心理念**：研究相移污染的本质，找到彻底解决污染、恢复城市的方法，支持探索核心污染区获取研究样本，反对工程幸存者无差别杀害所有进入核心区的人
- **势力范围**：城西高校区、周边科技园区，管控相对宽松，允许携带低污染样本进入营地（需经过消毒）

### 专属生产链&独有物品
- 专属产出：高级抗污染药剂、电子元件、改装装备配件、污染研究报告
- 独有配方：
  1. 长效抗污染药剂：效果是普通药片的3倍，可在中度污染区停留2小时无损伤
  2. 高级污染探测仪：探测范围扩大到200米，可精准定位隐藏畸变体、稀有物资位置
  3. 污染中和试剂：可小范围中和重度污染区域，开辟临时安全通道
- 独有武器：激光笔改装的激光枪、实验室化学品制作的腐蚀喷射器，对所有类型畸变体伤害加成20%

### 声望奖励
| 声望等级 | 解锁权限 |
| --- | --- |
| 敌对 | 进入高校区范围会被主动攻击，会被科技营发布悬赏通缉 |
| 中立 | 可在校区交易区兑换基础电子配件、普通药剂 |
| 友善 | 可接科研相关任务，可进入实验室帮忙处理样本获取高额奖励 |
| 尊敬 | 解锁所有独有配方，可兑换高级探测仪、长效药剂 |
| 崇拜 | 解锁「拯救结局」：和科研团队一起研究出彻底封堵污染裂隙的方法，修复整个CDC的污染，成为全民英雄 |

### 剧情关联
- 主线关键节点：科技营掌握着污染核心的所有研究数据，想解开污染真相、走拯救结局必须和科技营合作
- 冲突点：科技营需要进入核心污染区获取样本，和工程幸存者直接敌对，玩家需要选择帮科技营突破工程幸存者的封锁，还是帮工程幸存者阻止科技营进入核心区

## 3. FN河游民帮派（参考：府南河沿岸拾荒者）
### 基础信息
- **选址**：FN河东岸沿线多个临时据点，没有固定核心营地，打一枪换一个地方
- **人员构成**：灾变前的拾荒者、外卖员、司机、小商贩，总人数约200人，是消息最灵通、机动性最高的阵营
- **核心理念**：利益至上，没有固定立场，谁给的好处多就和谁合作，靠拾荒、跑商、卖情报为生，游走在各个阵营之间做中间商，不主动和任何阵营敌对，也不深度参与任何阵营的冲突
- **势力范围**：FN河全域、城区所有低/中度污染区，哪里有物资就去哪里，熟悉所有城市小路和隐藏物资点

### 专属生产链&独有物品
- 专属产出：稀有物资、隐藏情报、跨阵营交易渠道、违禁物品
- 独有权益：
  1. 全地图隐藏物资点坐标：只有游民阵营知道的未被搜刮过的高价值物资点
  2. 跨阵营交易权限：可帮玩家买到其他阵营不对外出售的禁售物品
  3. 全局情报：所有阵营的动向、即将发生的随机事件预警
- 独有武器：改装弩、毒针发射器、燃烧瓶，攻速快，适合偷袭、跑路

### 声望奖励
| 声望等级 | 解锁权限 |
| --- | --- |
| 敌对 | 所有游民都会拒绝和你交易，还会把你的位置卖给敌对阵营 |
| 中立 | 可进行基础交易，购买普通物资、低价值情报 |
| 友善 | 可接跑商、护送任务，可购买中价值情报、隐藏物资点坐标 |
| 尊敬 | 解锁跨阵营交易权限，可购买禁售物品、高级情报（阵营动向、事件预警） |
| 崇拜 | 解锁「巨贾结局」：成为游民帮派的老大，垄断所有阵营的交易渠道，掌控整个CDC的物资流通，富可敌国 |

### 剧情关联
- 主线关键节点：所有隐藏线索、稀有物资的位置都只有游民知道，想快速收集资源、解锁隐藏剧情必须和游民合作
- 冲突点：游民帮派内部有派系冲突，一派想和工程幸存者合作封锁核心区，一派想和科技营合作倒卖核心区样本，玩家需要选择支持哪一派

## 4. 工程守望者（原地铁施工队幸存者）
### 基础信息
- **选址**：M1线北段未被污染的高架站点，临近核心污染区
- **人员构成**：灾变时的M1线TZL站施工队幸存者、安保人员，总人数约80人，是战斗力最强的阵营
- **核心理念**：经历过污染爆发的全过程，亲眼看到接触核心污染的人全部畸变，坚信「任何进入核心污染区的人都会把污染带出来扩散」，会无差别攻击所有试图进入核心区的人，哪怕是无辜幸存者，目的是用自己的方式阻止污染进一步扩散，带有强烈的悲剧色彩
- **势力范围**：M1线北段、核心污染区周边3公里范围，管控极其严格，无关人员靠近直接攻击

### 专属生产链&独有物品
- 专属产出：重型改装武器、施工工具、混凝土/钢筋建材、污染阻隔材料
- 独有配方：
  1. 污染阻隔炸弹：唯一能对最终BOSS核心坍缩畸胎造成有效伤害的武器
  2. 重型混凝土防具：防御是普通防刺甲的3倍，可抵御重型畸变体的碾压攻击
  3. 工程探测仪：可探测地下结构、隧道坍塌风险，是进入核心区隧道的必备道具
- 独有武器：电钻改装的穿刺武器、大锤改装的重武器、切割机改装的近战武器，对金属/混凝土类畸变体伤害加成50%

### 声望奖励
| 声望等级 | 解锁权限 |
| --- | --- |
| 敌对 | 只要靠近核心区3公里范围就会被主动攻击，是前期最难对付的敌人 |
| 中立 | 不会主动攻击你，可在营地门口交易建材、重型武器配件 |
| 友善 | 可接巡逻、封锁任务，可进入营地休整，购买重型武器 |
| 尊敬 | 解锁所有独有配方，可兑换污染阻隔炸弹、核心区隧道通行权限 |
| 崇拜 | 解锁「守护结局」：成为工程守望者的领袖，和他们一起永久封锁核心污染区，阻止污染向外扩散，守护所有幸存者的安全 |

### 剧情关联
- 主线关键节点：只有工程守望者知道核心污染区的内部结构、最终BOSS的弱点，想走守护结局、打最终BOSS必须获得他们的信任
- 冲突点：他们和科技营是死敌，玩家选择帮科技营就会和他们永久敌对，想解锁真结局需要找到双方的平衡点，说服他们合作
