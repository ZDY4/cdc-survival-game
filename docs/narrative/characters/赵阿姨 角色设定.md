# 赵阿姨 角色设定
### 基础信息
- 身份：原社区网格员，幸存者据点01民生管理员
- 常驻位置：据点办公区
### 核心背景
做了15年网格员，周边小区的每户人家情况她都记得清清楚楚，灾变后主动承担了据点的民生管理工作，统计物资、安排住宿、协调矛盾，把据点管理得井井有条，大家都很服她。
### 关联内容
- 绑定任务：寻人、物资征集、帮助其他幸存者类任务
- 线索产出：失踪幸存者位置、居民家里隐藏物资点
- 交互逻辑：喜欢吃火锅底料煮的菜，玩家给她带火锅底料的话会给玩家额外的贡献点奖励，好感度达标后会告诉玩家哪些居民家里藏了好东西，还会给玩家安排条件更好的住宿房间。
