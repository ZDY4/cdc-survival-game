# 商人老王 角色设定
### 基础信息
- 身份：原**SX桥（参考：送仙桥）**古玩市场摊主，幸存者据点01交易负责人
- 常驻位置：幸存者据点01室内交易区
### 核心背景
多年前在送仙桥摆摊做古玩生意，灾变发生时第一时间带着店里和家里囤积的大量生存物资逃到据点，主动承担了物资交易和调度的工作，对各类物资的价值了如指掌，是据点里最受信任的人之一。
### 关联内容
- 绑定任务：全品类交易任务、道具收集类支线
- 线索产出：稀缺物资交易渠道、隐藏物资点位置
- 交互逻辑：可收购玩家搜集的闲置物资、出售生存补给与武器改装配件，好感度达标后解锁稀缺物资专属购买权限，还会告诉玩家哪里能找到高价值的古董藏品（可兑换大量贡献点）
