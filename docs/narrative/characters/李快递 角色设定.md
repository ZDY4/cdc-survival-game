# 李快递 角色设定
### 基础信息
- 身份：原**SF同城（参考：顺丰同城）**快递员，幸存者据点01探路员
- 常驻位置：据点门口休息区
### 核心背景
灾变前每天跑100多单，对**CDC（参考：成都）**三环内的所有小路、小区、商铺位置了如指掌，灾变后第一时间靠熟悉的路线躲过了污染扩散，是据点里外出存活率最高的人，所有外出队伍的路线都要先问过他。
### 关联内容
- 绑定任务：探路、跑腿、情报收集类任务
- 线索产出：城区安全路线、隐藏物资点位置
- 交互逻辑：会和玩家聊他之前送快递遇到的趣事，好感度达标后会告诉玩家只有快递员知道的近路和隐藏物资点，还会给玩家定制专属最优路线。
