# 张师傅 角色设定
### 基础信息
- 身份：原小区门口修电瓶车的师傅，幸存者据点01机械师
- 常驻位置：据点维修车间
### 核心背景
修了20年电瓶车，任何电动、机械故障看一眼就知道问题在哪，灾变后靠手艺给大家修武器、改装备，把电瓶车电机改成了电动陷阱，把头盔改成了防污染面罩，是据点里最受欢迎的人。
### 关联内容
- 绑定任务：收集机械零件、维修装备、改装武器类任务
- 线索产出：武器改装蓝图、机械类道具获取位置
- 交互逻辑：喜欢喝白酒，玩家给他带白酒的话会免费给玩家改装装备，好感度达标后会给玩家定制专属改装武器，比如带电击的工兵铲。
