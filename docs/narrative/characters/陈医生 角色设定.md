# 陈医生 角色设定
### 基础信息
- 身份：原**HX医院（参考：华西医院）**内科医生，幸存者据点01专属医师
- 常驻位置：幸存者据点01室内医疗区
### 核心背景
灾变前是首批接诊相移污染异常病例的一线医生，保留了完整的「患者生理指标完全正常、但认知彻底混乱，反复念叨墙里有眼睛」的接诊记忆，是玩家解锁污染机制、「爆发之初」剧情线的核心NPC。
### 关联内容
- 绑定任务：「寻找药品」、所有医疗类交互任务
- 线索产出：污染机制相关线索、灾变初期医疗记录碎片
- 交互逻辑：可兑换医疗物资、治疗玩家伤势，好感度达标后解锁核心剧情线索
